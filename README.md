# 📊 AI Money Mentor

AI Money Mentor is an intelligent personal finance assistant that helps users manage their income, expenses, savings, and financial goals through an interactive conversational interface. The system analyzes financial inputs and generates structured recommendations such as budgeting strategies, savings plans, expense optimization tips, and beginner-friendly investment guidance.

This project is designed especially for students, fresh graduates, and early-career professionals who want simple and practical financial planning support.

---

# 🚀 Features

## ✅ Personalized Budget Planning
Generates structured monthly budget suggestions based on income and spending patterns.

## 💰 Savings Strategy Recommendations
Suggests achievable saving targets and emergency fund planning strategies.

## 📉 Expense Optimization Tips
Identifies unnecessary spending habits and recommends corrections.

## 🎯 Goal-Based Financial Planning
Helps users plan purchases such as laptops, education expenses, or emergency funds.

## 📊 Beginner-Friendly Investment Guidance
Provides safe and practical investment awareness suitable for beginners.

## 🤖 Conversational AI Interface
Users interact naturally through a chat-based assistant for financial guidance.

---

# 🧠 How It Works

## Step 1: User Input
Users provide financial details such as:

- Monthly income
- Expenses
- Savings
- Debt (if any)
- Financial goals
- Risk tolerance level

## Step 2: Financial Analysis
The AI system analyzes the financial condition using structured prompts and decision logic.

## Step 3: Smart Recommendations
The assistant generates:

- Budget allocation suggestions
- Savings improvement strategies
- Expense reduction tips
- Investment awareness guidance
- 30-day financial improvement action plan

# ⚙️ Installation Guide

## Step 1: Clone the Repository
git clone https://github.com/GayatriMusunuri/AI-Money-Mentor

## Step 2: Navigate to Project Directory
cd ai-money-mentor 

## Step 3: Install Dependencies
npm install 

## Step 4: Configure Environment Variables

Create a file named:
.env.local

Add your API key:
GEMINI_API_KEY=your_api_key_here

## Step 5: Run the Development Server
npm run dev

The application will run at:
http://localhost:3000/
